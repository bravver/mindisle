# 心屿 MindIsle

> 每一座孤岛，都值得被看见

大学生心理状态可视化与互助平台，通过温暖治愈的视觉风格，为大学生提供情绪记录与互助陪伴的工具。

## 功能模块

- **心情气象站** — 7种天气心情打卡 / 日历热力图 / 心情故事时间线
- **情绪山海** — Canvas三色山脉可视化，实时解读压力/焦虑/孤独
- **汐岛** — 生理周期关怀，根据周期阶段定制饮食/运动/情绪建议
- **思绪钓鱼** — 黄金矿工风格减压游戏，随机鱼获+励志寄语
- **互助树洞** — 匿名发帖，💡点亮鼓励机制

## 技术栈

- 纯前端（HTML + CSS + JavaScript）
- Canvas API（动态山脉、游戏画面）
- LocalStorage 数据持久化
- 无后端依赖
